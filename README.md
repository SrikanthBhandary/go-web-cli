# go-web-cli

## Install: ##
``` 
go install github.com/SrikanthBhandary/go-web-cli
```



## Usage: ##
```
go-web-cli --pkgname <pkgname>
```

<img src="folder_structure.png"/>

## Run ##

```
cd app
go run cmd/todo.go 
```

Now open the browser and launch the url  http://127.0.0.1:8080/todos
